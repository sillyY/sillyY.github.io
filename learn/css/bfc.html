# BFC

# 简介

BFC（Block Formatting Context）格式化上下文，是 Web 页面中盒模型布局的 CSS 渲染模式，指一个独立的渲染区域或者说是一个隔离的独立容器。

# 如何触发 BFC

  • <html>根元素
  • float 的值不为 none。
  • overflow 的值为 auto,scroll 或 hidden。
  • display 的值为 table-cell, table-caption, inline-block 中的任何一个。
  • position 的值不为 relative 和 static。

# BFC 特性

  • 内部的 Box 会在垂直方向上一个接一个的放置。
  • 垂直方向上的距离由 margin 决定
  • bfc 的区域不会与 float 的元素区域重叠。
  • 计算 bfc 的高度时，浮动元素也参与计算
  • bfc 就是页面上的一个独立容器，容器里面的子元素不会影响外面元素。

# 特性 1

与正常规则一致，不阐述解释

# 特性 2

不发生 margin 合并.

  • 两者的 margin 为相同符号
    • 都是正数，取最大正数, 如: 30px 和 50px, 取 50px
    • 都是负数，则最小数，如：-30px 和 -50px， 取 -50px
  • 两者的 margin 为不同符号 取两者之和，如: -30px 和 50px，取 20px

# 特性 3

可以此实现两栏布局，或三栏布局

# 两栏布局

<aside></aside>
<main></main>

aside {
    float: left;
    width: 300px;
    height: 500px;
    margin-right: 10px;
    background-color: red;
}
main {
    overflow: hidden;
    height: 500px;
    background-color: green;
}

# 三栏布局

<div class="left"></div>
<div class="right"></div>
<div class="middle"></div> // 注意middle位于最后

.left {
    float: left;
    width: 100px;
    height: 300px;
    background-color: green;
}
.right {
    float: right;
    width: 200px;
    height: 300px;
    background-color: blue;
}
.middle {
    overflow: hidden; /*创建bfc*/
    height: 300px;
    background-color: red;
}

# 特效4

利用overflow:hidden清除浮动
由于浮动元素无法撑出处于标准文档流的父盒子的height，可使其形成BFC，从而实现height计算

# 特效5

隔离的独立容器
