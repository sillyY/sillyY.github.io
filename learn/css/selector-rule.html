# CSS选择器优先级规则

# 0级

通配符选择器、选择符和逻辑组合伪类。

# 通配符选择器

通配符选择器写作星号(*)。示例如下：

* { color: #000;}

# 选择符

  • +选择符 相邻兄弟选择符

匹配后面的第一个兄弟元素

.cs-li + li {
    color: skyblue;
}
  • ~选择符 随后兄弟选择符

匹配后面的所有兄弟元素

.cs-li ~ li {
    color: skyblue;
}
  • 空格选择符 后代选择符
.cs-module-x p {
    color: #666;
}
  • || 选择符 列选择符

选择所有属于col.selected的<td>, 哪怕这个<td>元素横跨多个列

col.selected || td {
    background-color: skyblue
}

# 逻辑组合伪类

这些伪类本身并不影响CSS优先级、影响优先级的时括号里面的选择器

  • :not() 用来匹配不符合选择器的元素

  • :is() 用来匹配符合选择器的元素

  • :where() 用来匹配符合选择器的元素

与:is()的含义、语法、作用一致，唯一区别为 :where()伪类的优先级永远是0, :is() 的优先级是由它的选择器列表中优先级最高的选择器决定的.

# 1级

标签选择器，示例如下：

body { color: #333; }

# 2级

类选择器、属性选择器和伪类，示例如下：

.foo { color: #666; }
[foo] { color: #666; }
:hover { color: #333; }

# 3级

ID选择器, 示例如下:

#foo { color: #999; }

# 4级

style属性内联，示例如下:

<span style="color: #ccc;">优先级</span>

# 5级

!important最高优先级, 示例如下:

.foo { color: #fff !important; }
