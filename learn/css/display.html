# 元素显示与隐藏

  • 元素不可见，同时不占据空间，辅助设备无法访问，同时不渲染
<script type="text/html">
  <img src="1.jpg">
</script>
  • 元素不可见、不占据空间、辅助设备无访问、但资源有加载、DOM 可访问
.dn {
  display: none;
}
  • 元素不可见、不占据空间、辅助设备无法访问、但隐显的时候有 transition 淡入淡出效果
.hidden {
  position: absolute;
  visibility: hidden;
}
  • 元素不可见、不能点击、辅助设备无法访问、但占据空间
.vh {
  visibility: hidden;
}
  • 元素不可见、不能点击、不占据空间、但键盘可访问，则可以使用 clip 剪裁隐藏
.clip {
  position: absolute;
  clip: rect(0 0 0 0);
}
.out {
  position: relative;
  left: -999em;
}

// clip-path 新增代替clip, 但兼容性不佳
.clip-path {
  clip-path: polygon(0px 0px, 0px 0px, 0px 0px, 0px 0px);
}
  • 元素不可见、不能点击，但占据空间，键盘可访问
.lower {
  position: relative;
  z-index: -1;
}
  • 元素不可见、能点击、不占据空间
.opacity {
  position: absolute;
  opacity: 0;
  // IE
  filter: Alpha(opacity = 0);
}
  • 文字隐藏: text-indent 缩进隐藏
  • transition 动画： max-height 隐藏
